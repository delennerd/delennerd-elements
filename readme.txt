You use bootstrap in your theme? Than use this plugin. It adds some bootstrap components without hundreds of div in divs.

== Description ==

Add bootstrap components to your elementor page.

*   Bootstrap Card
*   Bootstrap Buttons as list
*   Section headline with subtitle (top or bottom)
*   Flipbox with background image or without

== FAQ ==

= Is the bootstrap.css and .js included in this plugin? =

No, you must implement it in your theme.
